# DNA Font Animate

## Локальный просмотр

### Вариант 1: открыть файл напрямую
Откройте `index.html` в браузере двойным кликом.

### Вариант 2: через локальный сервер
Запустите сервер из корня репозитория:

```bash
python -m http.server 8000
```

Затем откройте в браузере:

```
http://127.0.0.1:8000/index.html
```
